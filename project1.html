Project Highlights

Time Frame:

1 Week

Role:

Web Developer

Objective:

Deliver a calming mental wellness tool promoting reflection and balance.

Design Process

User Discovery

Surveys, interviews, and insights from psychology research.

UX Design

Sketches, user flows, wireframes (low-fi to hi-fi in Figma).

Prototyping & Testing

Moderated usability tests, heuristic reviews, iterations.

Build

HTML/CSS/Bootstrap responsive interface with accessibility focus.
